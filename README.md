# GitHub Pages Plot Browser

Static site for browsing the per-sequence time-series plots under `analysis/separate_time_plots/`.

## Contents

- `index.html`: main viewer
- `styles.css`: site styling
- `app.js`: selector/filter logic
- `assets/plots/`: copied PNG/PDF plot assets
- `assets/data/plots.json`: plot manifest

## Deploy

1. Create a GitHub repository and copy the contents of `github-pages-site/` into its root.
2. Push to GitHub.
3. In the repository settings, enable GitHub Pages from the default branch root.

This site uses only static files, so it works on standard GitHub Pages without a build step.
